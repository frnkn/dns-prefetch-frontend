DNS Prefetching: Simply Create DNS Prefetch Meta Tags From Your Website to Make Your Site Load Faster

DNS Prefetching enables your users to load CSS, Javascript and images faster. The X-DNS-Prefetch-Control HTTP response header controls DNS prefetching, a feature by which browsers proactively perform domain name resolution on both links that the user may choose to follow as well as URLs for items referenced by the document, including images, CSS, JavaScript, and so forth.
